# PVZ
- 左键点击植物卡片选择植物，点击地块种下
- 点击右上角铲子铲除植物
- 豌豆射手若检测到前方有僵尸则发射子弹
- 僵尸走到最后一格则走进房屋，游戏结束
- 采用缓存池管理游戏中的对象，包括游戏中的植物、僵尸、阳光、僵尸死亡的尸体与掉落的头部特效
